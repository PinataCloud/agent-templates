# 🧱 Basic

Starter templates that provide a vanilla foundation for building any
kind of Hermes agent.

## Use this for
- Getting started with Hermes agents
- Learning the flat layout (`SOUL.md`, `AGENTS.md`, `manifest.json`,
  `skills/`)
- Scaffolding a new agent from scratch

## Example Templates
- **[Useful Assistant](./useful-assistant)** — A vanilla persona with
  workspace conventions. No pre-attached skills — add what you need.

## Creating a Template

To add a template to this category, create a new folder here with your
template files and submit a PR.
