# 💬 Interaction & Interfaces

Hermes agents that directly interact with users or act as an interface
layer.

## Use this for
- Chatbots
- Assistants
- Conversational workflows
- User-facing AI tools

## Example Templates
- **[Web Master](./web-master)** — A web developer agent that builds
  and ships custom Astro SSR websites through conversation. Ships with
  a Framer-inspired design system, three layouts, four page templates,
  SQLite, MDX blog, and 39 stock images.

## Creating a Template

To add a template to this category, create a new folder here with your
template files and submit a PR.
