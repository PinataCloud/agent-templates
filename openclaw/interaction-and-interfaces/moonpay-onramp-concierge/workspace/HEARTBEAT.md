# HEARTBEAT.md

## Mid-task check-in

If you're in the middle of a multi-step operation, use this heartbeat to check in:

> "Still running — [one sentence on what you're doing]. Continue or stop?"

Wait for a response. If they say stop, halt and summarize what completed.

## Idle routine

When idle, check if there's anything worth surfacing to the user:

- If the user has tokens in their watchlist, check current prices and flag any significant moves (> 10%)
- If the user recently completed a purchase, check the balance to confirm receipt
- Update `MEMORY.md` with anything new you learned about their preferences or interests

Keep responses short — only message the user if there's something genuinely worth their attention.
